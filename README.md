## Bu bölüm ne ile ilgili?

Buradaki amaç, circle_area isimli bir function yazmak.
Bu function tek bir parametre alır: radius.
Görevi ise verilen yarıçapa sahip bir dairenin alanını döndürmektir.

Hatırlatma olarak formül şöyleydi:

A = pi * r ^ 2

demo.py dosyasını aç ve bunu oraya uygula!

Terminalden make komutunu çalıştırarak koşabileceğin 3 test bulunuyor.
Bir sonraki göreve geçmeden önce tüm testlerin yeşil olduğundan emin ol!
